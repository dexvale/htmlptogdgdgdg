Positioning of elements in CSS

Positioning the elements of an HTML page is no easy feat. CSS provides a set of methods that can be used cumulatively:

  • Margin and spacing: via the margin and paddiing properties
  • Centering: vertical or horizontal via the text-algin a vertical-align properties
  • Display Type: display property
  • Positioning: position property
  • Floating Object: float property

1. Margin and Spacing

Margin and Spacing are used as priority a block with respect to another block or with respect to its parent block. The margin property is used to apply outer spacing to block and the padding property applies inner spacing to the block.

A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border) wrapped in a parent block (blue border) and vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels. 

    <div style="border: 2px solid blue;">
      <p style="padding: 5px; margin: 15px 10px; border: 1px dashed gray">
        A paragraph of text with an inner spacing of 5 pixels all around the text (gray dotted border) wrapped in a parent block (blue border) and
          vertical outer spacing of 15 pixels and horizontal outer spacing of 10 pixels. 
        </p>
    </div>

2. Horizontal and Vertical centering

2.1 Horizontal Centering

To center a text horizontally, the text-align property wll be used. To cneter a block horizontally, one method is to use the automatic margins technique (margin: auto -cf. above. Another is to make use a flexible boxes with the justify-content property.

A paragraph of text with a width of 50% that of the parent block, centerd horizontally, and whose text is also centerd horizontally. 

                <div style="border: 2px solid blue;">
                    <p style="text-align: center; width: 50%; margin: auto; border: 1px dashed gray;">
                    A paragraph of text with a width of 50% that of the parent block, centerd horizontally, and whose text is also centerd horizontally.
                </p>
          </div>

2.2 Vetical Centering

Vertical centering will depend on the container element. If the container block has the display: table-cell property, such as a tabel for example, you can use vetical-align: middle Otherwise you can use the flexible boxes and the align-items property

Text paragraph 50% wide, centerd vertically and horizontally in its parent block. 

                <div style="border: 2px solid blue; height: 250px; width:  500px; display: flex; align-items: center; justify-content: center;">
                    <p style="text-align: center; width: 50%; border: 1px gray dashed;">
                    Text paragraph 50%  wide, centerd vertically and horizontally in its parent block.
                </p>
          </div>

3. Dispaly Type

Changing the rendering mode of HTML elements via the display property enables easy positioning of the elements relative to one anoter. Display Block / Inline - Inline - block - Table - cell - Flex (spacebetween)

3.1 Display Block / Inline

By default, the main HTML elements use the block (dic, p...) or inline (a, span...) rendering. It is possible to modify the rendering mode of an element to change its display mode. Example. use a bullet list containning links to generate a horizontal menu

  • Home
  • Section 1
  • Section 2
  • Section 3
  • Contact
                    <nav>
                        <ul class="menu">
                        <li><a href="">Home</a></li>
                        <li><a href="">Section 1</a></li>
                        <li><a href="">Section 2</a></li>
                        <l><a href="">Section 3</a></li>
                        <li><a href="">Contact</a></li>
                    </ul>
                </nav>

3.2 Display Inline-Block / Table-cell...

Other types of rendering modes allow, for example, inline and block display be mixed: inline-block which alows a width, a height, margins and spacing to be addded to an inline block. Table-cell mode allows table properties to be applied to a block, generally used to center vertically. 

                    <div>
                        <label style="display: inline-block; width: 280px; color: green;">The inline-block label in my field</label>
                        <input type="email" name="email" placeholder="Your email">
                </div>

3.3 Display Flex

It would take several laboratory exercise to cover all the possibilities presented by flexible boxes. This modernn and very powerful rendering mode make it possible to easily position elements relative to each other. 

                <div style="display: flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
                  <div style="width: 30%;  margin: 1.5%; border: 1px solid green></div>
                  <div style="width: 30%; margin: 1.5%; border: 1px solid pink"></div>
                  <div style="width: 30%; margin: 1.5%; border: 1px solid grey"></div>
              </div>

Positioning Type

By default, all elements in the HTML flow are positioned statically =>position:static It is possible to position elements by Changing the value of the position property

4.1 Fixed Position

Fixed positioning allows an element to betaken ouyt of the HTML flow and placed in a specific location by adjusting the top, bottom, right, and left properties. This is the case, for example, of the header of the page, which is fixed and which has the following CSS style. 

                .header-fix{
                      height: 85px;
                      width: 100%;
                      position: fixed;
                      top: 0;
                      left: 0;
                      background-color: #eee;
                      border-bottom: 1px solid #ddd;
                      box-shadow: 0px 3px 5px 0px rgba(0, 0, 0, 0.75);
                  }

4.2 Relative Position

Relative positioning keeps the element in the flow but allows it to be offset from its initial position via left, right, top and bottom properties.

This block is offset by 10 pixels down and told the right 
                <div style="display: flex; justify-content: space-between; border: 2px solid blue; height: 80px;">
             <div style="width: 30%; margin: 1.5%; border: 1px solid green;"> </div>
             <div style="width: 30%; margin: 1.5%; border: 1px solid pink; position: relative;  top: 10px; left: 10px;">This block is offset by 10 pixels
              down and told the right
             </div>
             <div style="width: 30%; margin: 1.5%; border: 1px solid gray;"> </div>
           </div>

4.3 Absolute Position

Absolute positioning takes the HTML element out of the flow and position it relative to the wrapping block(having the relative positioning property) according to the values provided by top, left, right, and / or bottom.

This block is offset by 10 pixels down and to the right with respect to the parent block 
                <div style="display: flex; justify-content: space-between; border: 2px solid blue; height: 80px; position: relative; " >
            <div style="width: 30%; margin: 1.5%; border: 1px solid green"></div>
            <div style="width: 30%; margin: 1.5%; border: 1px solid pink; absolute: relative; top: 10px; left: 10px;">
              This block is offset by 10 pixels down and to the right with
              respect to the parent block </div>
            <div style="width: 30%; margin: 1.5%; border: 1px solid gray"></div>
          </div>
