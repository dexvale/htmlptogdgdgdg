Positioninig of Elements in CSS

Positioning thew elements of an html page is no easy feat. CSS provides a set of methods that can be used cumulatively

  • Margin and spacing :via the margin and padding properties
  • Centering vertical or horizontal via the text-align and vertical-align properties
  • Display Type display property
  • Positioning: positioning

1. Margin and Spacing

Margin and Spacing are used as a priority to position a block with respect to another block or with respect to its partner block. The Margin property is used to apply outer spacing to the block and the Padding property applies inner spacing to the block
